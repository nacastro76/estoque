# Sistema de Controle de Estoque

Sistema desenvolvido em React para controle de estoque de materiais e projetos.

## Funcionalidades

- Cadastro de materiais
- Controle de entrada e saída
- Alertas de baixo estoque
- Dashboard resumido
- Busca de materiais

## Tecnologias

- React
- Vite
- JavaScript

## Como executar

```bash
npm install
npm run dev
```
